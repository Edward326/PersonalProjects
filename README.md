## Short description
* This folder demonstrates the knowledge accumulated and applied in the field of the IT industry, both from the hardware and software product development perspectives.
* The projects contained in this folder take the theory learned from university, more exactly from the computer engineering major, and put them into practice.
* The repository contains projects that are done as assignments for different subjects from school, such as Basic Concepts of Artificial Intelligence, Object-Oriented Programming, or Operating System Concepts. It also contains projects developed in free time, from a passion to automate processes and raise productivity.

## Repository structure
* **HW** - projects that relate to implementing simple to medium functionality hardware components, to understand the facilities and architectures behind them.
* **SW** - applications that cover various IT fields, such as an application that uses AI to empower spatial awareness and indoor navigation for low-eyesighted persons, or even blind individuals, or applications that experiment with different configurations for models (changes in architecture, training specifications), for further deployment in different areas (identification of indoor and outdoor objects at pixel levels, e.g., Semantic Segmentation task-oriented model). Also, we have products in the field of Web Development that implement both frontend and backend needs for a fully operational website.
